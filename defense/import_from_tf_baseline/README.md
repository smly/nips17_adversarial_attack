## Usage

```
$ python ens_adv_inception_resnet_v2.py \
    --checkpoint /data/pretrained/ens_adv_inception_resnet_v2.ckpt \
    --outdir /tmp/ens_adv_inception_resnet_v2
```

* numpy
* click
* Pillow
* h5py
